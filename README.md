# patient-test-task

* .NET 6

### Before testing
1. Build application. Open a terminal in the project root and execute command:
>```shell
>docker-compose build
>```

2. Create docker containers. In terminal execute command:
>``` shell
>docker-compose up -d
>```

3. Wait for the migrations to be applied.
